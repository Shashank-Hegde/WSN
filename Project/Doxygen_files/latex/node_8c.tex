\doxysubsection{Detailed Description}
Wireless nodes code for player devices in the W\+SN Lab project. 

This file contains the implementation of wireless nodes used to track the health and performance of athletes in real-\/time. The nodes communicate with each other and a central gateway through wireless sensor networks.
