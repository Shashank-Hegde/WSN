Leveraging Wireless Sensor Networks for Real-Time Health and Performance Tracking of Athletes  0.1
The WSN Lab project aims to utilize wireless sensor networks to track the real-time health and performance of athletes. By employing sensors and data reporting techniques, the project enables monitoring of vital parameters such as heart rate, oxygen saturation (SpO2), and force exerted during physical activities. The collected data is analyzed and presented through a graphical user interface (GUI) for trend analysis, personalized alerts, and historical data analysis. The project also incorporates jump stress analysis using pressure sensors to analyze leg stress during jumps, providing insights for injury prevention and performance optimization.
